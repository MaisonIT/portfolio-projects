# Frontend Developer Portfolio | Михайло Петрушов

Вітаю! Мене звати Михайло. Я — Frontend / Markup розробник. У цьому репозиторії зібрано мої ключові роботи: від реальних комерційних проєктів та кастомних вебдодатків до інтерактивних пет-проєктів та експериментів з UI/UX.

Моя мета в кожному проєкті — писати чистий, семантичний код, вибудовувати масштабовану архітектуру стилів та створювати бездоганний адаптив (Mobile-First) з великою увагою до деталей та продуктивності сайту.

* **Frontend:** HTML5 (Семантична верстка), CSS3 / SCSS / Less (Методологія BEM)
* **Логіка & Дані:** JavaScript (ES6+, Робота з DOM, Fetch API, JSON)
* **Компонування:** Flexbox, CSS Grid, Pixel-Perfect верстка з макетів Figma
* **Інструменти & Робочий процес:** Git / GitHub, Оптимізація швидкості (Web Performance), Адаптивний та кросбраузерний дизайн

## 🚀 Найцікавіші проєкти

### 🛒 Cyber (E-commerce Web Application)
Повноцінний, швидкий інтернет-магазин техніки з динамічною логікою без використання важких фреймворків.
* **Технології:** HTML5, SCSS (BEM), Vanilla JavaScript, JSON
* **Фічі:** Динамічний рендеринг карток товарів з бази даних JSON, повністю робоча інтерактивна корзина (додавання, видалення, менеджмент кількості та миттєвий перерахунок суми в реальному часі), розумна система фільтрації та сортування товарів без перезавантаження сторінки.
* **🔗 Live Demo:** [Переглянути проєкт](https://maisonit.github.io/portfolio-projects/e_store_cyber/)

### 🧹 Cleaning Web Service
Реальний комерційний вебсайт для клінінгового сервісу, оптимізований під бізнес-завдання.
* **Технології:** HTML5, CSS3, JavaScript, Кроссбраузерна верстка
* **Фічі:** Pixel-perfect адаптація під усі типи мобільних пристроїв, висока швидкість завантаження для кращого SEO, чиста та структурована кодова база.
* **🔗 Live Demo:** [Переглянути проєкт](https://dpkliduklid.cz/)

### 🏢 Creative Agency Website
Сучасний корпоративний лендінг з акцентом на преміальну типографіку, мікроінтеракції та плавні анімації.
* **Технології:** HTML5, SCSS, JavaScript
* **📁 Код:** `/creative-agency` | **🔗 Live Demo:** [Переглянути проєкт](https://maisonit.github.io/portfolio-projects/creative_agency/)
